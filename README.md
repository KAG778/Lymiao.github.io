# 吴彦彦 | Academic Homepage

Personal academic homepage built with vanilla HTML, CSS, and JavaScript.

## Features

- Bilingual support (Chinese / English)
- Responsive design
- Clean academic profile layout
- Blog section (coming soon)

## Structure

```
├── index.html          # Main page
├── css/main.css        # Stylesheet
├── js/main.js          # JavaScript (i18n, navigation)
├── assets/avatar.jpg   # Profile photo
└── README.md
```
